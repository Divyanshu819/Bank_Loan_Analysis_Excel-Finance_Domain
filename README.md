#  Bank Loan Analytics Dashboard

##  Project Overview
The **Bank Loan Analytics Dashboard** is an interactive Excel project designed to analyze bank loan data and provide insights into lending performance, borrower behavior, and loan portfolio health.  

This project helps financial institutions make **data-driven lending decisions** by tracking key loan metrics, monitoring loan quality, and identifying trends in loan applications, funding, and repayments.

---

##  Tech Stack
The dashboard was built using the following tools and technologies:

-  **Microsoft Excel** – Data analysis and dashboard creation  
-  **Excel Dataset** – Bank loan data used as the primary data source  
-  **Pivot Tables & Pivot Charts** – Data aggregation and analysis  
-  **Slicers & Filters** – Interactive dashboard controls  
-  **Data Visualization** – Charts and KPIs for clear insights  

---

##  Data Source
**Source:** Bank Loan Dataset containing loan application and borrower information.
- Inspiration - Data Tutorials

The dataset includes:

- Loan applications and issue dates  
- Loan amount funded and total amount received  
- Interest rate and Debt-to-Income ratio (DTI)  
- Loan status (Fully Paid, Current, Charged Off)  
- Borrower information (state, employment length, home ownership)  
- Loan term and loan purpose  

---

##  Features / Highlights

### • Business Problem
Banks manage thousands of loan applications, making it difficult to track loan performance and identify risk patterns from raw data.  

Without proper analysis, it becomes challenging to:
- Monitor lending performance  
- Identify risky loans  
- Understand borrower profiles  
- Optimize lending strategies

---

### • Goal
To build an **interactive loan analytics dashboard** that:

- Tracks key loan performance indicators  
- Distinguishes between **Good Loans and Bad Loans**  
- Monitors monthly lending trends  
- Analyzes borrower characteristics and loan purposes  
- Provides insights for improving lending strategies  

---

### • Walkthrough of Key Visuals

#### Dashboard 1 – Summary
Displays key KPIs to monitor lending performance:

- **Total Loan Applications**
- **Total Funded Amount**
- **Total Amount Received**
- **Average Interest Rate**
- **Average Debt-to-Income Ratio (DTI)**

Also includes analysis of **Good Loans vs Bad Loans** based on loan status.

---

#### Dashboard 2 – Overview
Provides visual insights into loan trends and borrower characteristics:

- **Monthly Trends by Issue Date (Line Chart)** – Tracks loan applications, funded amount, and repayments over time  
- **Regional Analysis by State (Map Chart)** – Shows lending activity across different states  
- **Loan Term Analysis (Donut Chart)** – Distribution of loans by term length (36 vs 60 months)  
- **Employee Length Analysis (Bar Chart)** – Impact of employment history on loan applications  
- **Loan Purpose Breakdown (Bar Chart)** – Reasons borrowers take loans  
- **Home Ownership Analysis (Tree Map)** – Loan distribution based on home ownership status  

---

#### Dashboard 3 – Details
A detailed data view that provides a **complete snapshot of loan records**, allowing users to explore borrower profiles, loan performance, and financial metrics in depth.

---

### • Business Impact & Insights

-  Monitor **loan portfolio health** using key financial KPIs  
-  Identify **Good Loans vs Risky Loans** for better credit risk management  
-  Understand **borrower behavior and loan demand trends**  
-  Support **data-driven lending decisions** for financial institutions
